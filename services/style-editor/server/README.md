# خادم محرر الأسلوب الشخصي

هذا هو الخادم الخلفي (Backend) لتطبيق محرر الأسلوب الشخصي. وهو مبني باستخدام Express.js ويوفر واجهة برمجة تطبيقات (API) لإدارة بيانات التدريب (أزواج النصوص).

## الإعداد والتشغيل

لتشغيل هذا الخادم على جهازك المحلي، اتبع الخطوات التالية:

### المتطلبات المسبقة

-   [Node.js](https://nodejs.org/) (إصدار 14 أو أحدث)
-   npm (يأتي مع Node.js)

### 1. تثبيت الاعتماديات

افتح الطرفية (Terminal) وانتقل إلى مجلد `server` ثم قم بتشغيل الأمر التالي لتثبيت الحزم المطلوبة:

```bash
npm install
```

### 2. تشغيل الخادم

بعد اكتمال التثبيت، قم بتشغيل الأمر التالي لبدء تشغيل الخادم:

```bash
npm start
```

### 3. التحقق

إذا تم كل شيء بنجاح، يجب أن ترى الرسالة التالية في الطرفية:

```
Server is running on http://localhost:3001
```

الآن الخادم يعمل ويستمع للطلبات على المنفذ `3001`. الواجهة الأمامية للتطبيق مهيأة للتواصل مع هذا العنوان تلقائيًا.

## نقاط النهاية (API Endpoints)

-   `GET /api/pairs`: لجلب جميع أزواج التدريب المحفوظة.
-   `POST /api/pairs`: لإضافة زوج تدريب جديد.
-   `POST /api/pairs/batch`: لإضافة مجموعة من أزواج التدريب دفعة واحدة.
-   `DELETE /api/pairs/:id`: لحذف زوج تدريب محدد باستخدام `id` الخاص به.

**ملاحظة:** يستخدم هذا الخادم حاليًا مصفوفة في الذاكرة لتخزين البيانات. هذا يعني أن أي بيانات تدريب تضيفها ستُفقد عند إعادة تشغيل الخادم. للتخزين الدائم، يمكنك تعديل هذا الخادم ليتصل بقاعدة بيانات مثل MongoDB, PostgreSQL, أو SQLite.
